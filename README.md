# Android Lab 8 - Каталог AAA игр

## Описание проекта
Приложение представляет собой каталог популярных AAA игр для ПК. Пользователь может просматривать карточки с играми, листать их с помощью кнопок навигации и читать описание каждой игры.

## Использованные элементы UI

### Material Design 3 компоненты:
- **Surface** - основной контейнер приложения, задающий фон
- **Card** - карточка для отображения информации об игре
  - Используется для группировки изображения, названия и описания
  - Добавляет визуальное выделение и тени
- **Button** - кнопки навигации "Назад" и "Вперед"
  - С возможностью отключения (enabled/disabled)
  - Обработчики нажатий для смены текущей игры
- **Text** - для отображения:
  - Названия игры (стиль headlineSmall)
  - Описания игры (стиль bodyMedium)

### Компоновка (Layout):
- **Column** - вертикальное расположение элементов в карточке
- **Row** - горизонтальное расположение кнопок навигации
- **Modifier** - для настройки:
  - `fillMaxSize()` - растяжение на весь экран
  - `padding()` - отступы между элементами
  - `weight(1f)` - распределение пространства

### Изображения:
- **Image** - отображение постеров игр
  - `contentScale = ContentScale.Crop` - обрезка изображения под размер
  - `height(450.dp)` - фиксированная высота
  - Загрузка через `painterResource` из ресурсов drawable

## Ресурсы:
- **strings.xml** - все текстовые ресурсы (названия игр и описания)
- **drawable** - изображения постеров игр (anime1.png - anime10.png)
